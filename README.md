# GvCameraSDK Distribution Package

Generated by scripts/package_sdk.ps1

## Build
- cmake --build build_release --config Release --target GvCameraSDK

## Build Samples
- cmake -S . -B build_samples
- cmake --build build_samples --config Release
- cmake --build build_samples --config Debug
- sample exe output: `build_samples/samples/bin/Release`
- sample exe output (Debug): `build_samples/samples/bin/Debug`

## Create dist_out (Recommended)
- `.\package_dist.ps1`
- output root: `dist_out`
- Release files: `dist_out/bin/Release`, `dist_out/lib/Release`
- Debug files: `dist_out/bin/Debug`, `dist_out/lib/Debug`

## Create dist_out (Manual)
- cmake -S . -B build_dist -DGVSDK_INCLUDE_SAMPLE_EXES_IN_INSTALL=ON
- cmake --build build_dist --config Release
- cmake --install build_dist --config Release --prefix dist_out
- cmake --build build_dist --config Debug
- cmake --install build_dist --config Debug --prefix dist_out
